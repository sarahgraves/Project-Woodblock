Now Playing

The Now Playing section of an iHeartRadio app should be quickly and obviously accessable from anywhere in the app. This section must contain controls for key functionality associated with controlling audio playback and interacting with it. It can also contain additional content and presentations which enhance the experience of listening to an iHeartRadio station.

Some content and finctionality is not appropriate for some devices and interfaces. For example, song lyrics and artist bios should not be viewable in the context of an automotive application due to safety concerns.

Live Player: Required Controls & Content

The following items are associated with controlling playback at a station level.

Station Name

This is sometimes referred to as Call Letters

Station Positioner

This is sometimes referred to as Station Descriptor. Station Positioner is not required for devices with limited screen real estate.

Station Logo

On devices with limited screen real estate, the station logo can be replaced by song album art after a short period of time.

Play/Stop Button

The Play/Stop button should have the single greatest amount of emphasis on the Now Playing screen.

Scan Button

The Scan button should always be placed close to the Play/Stop button

The Scan button should be disabled if a Live station is playing that does not have scan logic.

Share Station Button

The Share station button should be closely associated with the station name + logo

The following items are associated with controlling playback on a song level.

Currently Playing Song Name

Currently Playing Artist Name

Currently Playing Album Art

On devices with limited screen real estate, the album art may replace the station logo after a short period of time.

On Air (DJ) Info

When song metadata is unavailable, such as during commercial breaks or talk segments, the "On Air" info should be displayed in place of the song info. For example: "On Air: Elvis Duran"

Create Custom Station Button

The user can create a custom station which uses the currently playing song as a seed. This button should be disabled any time song metadata is not available, e.g., during commercials or talk segments.

When the user clicks the Create Station button, the app should display a modal which reads "[seed song name] Radio has been created. Would you like to listen now?"

If the user clicks the Create Custom Station button during a song for which we can not create a custom station, the app should display a temporary message reading "Sorry, we cannot create a custom station from this song.

Thumbs Up & Thumbs Down Buttons

Thumbs Up and Down buttons should be available whenever song metadata is available.
***Additional information regarding Live Thumbs will be provided.***

Buy Song Button

Buy Song should be available on any platform that has a mechanism for purchasing music digitally.

Song Lyrics Button

Song lyrics do not necissarily need to be hidden behind a button. On platforms that allow for it, the first two lines of song lyrics may be displayed with a "More" link to view the full lyrics.

When the user calls up song lyrics by hitting the button the following information should be displayed:

  • Song Name
  • Artist Name
  • Album Name (not required)
  • Song Lyrics (with paragraph breaks)
  • An attribution to LyricFind that reads "Lyrics provided by LyricFind Inc."

If the user hits the Song Lyrics button when lyrics are not available, a temporary message should be displayed that reads "We couldn't find lyrics for this song."

Custom Player: Required Controls & Content

The following items are associated with controlling playback at a station level.

Station Name

This should be the name of the station; e.g., "Lada Gaga" for artist based radio, "Bad Romance" for song based radio, or "Summer Party" for iHeartRadio Originals; followed by the word "Radio" set in a distinguishing typeface.

Play/Pause Button

The Play/Pause button should have the single greatest amount of emphasis on the Now Playing screen.

Seek Button

The seek button should altways be placed close to the Play/Pause button

Discovery Tuner

The discovery tuner is a branded iHeartRadio element. It allows the user to adjust the amount of less familiar music that will play on a Custom station. The Discovery Tuner has three settings: "Familiar", "Mixed", and "Less Familiar". Each setting should display a list of 4 or 5 artists who's music will be featured on that station. This list should be titled "Featuring Music From:".

The following items are associated with controlling playback on a song level.

Currently Playing Song Name

Currently Playing Artist Name

Currently Playing Album Art

Elapsed Track Time / Total Track Time

Display the Elapsed Track Time in minutes:seconds followed by the Total Track Time in minutes:seconds seperated by a foward slash.

Create Custom Station Button

The user can create a custom station which uses the currently playing song as a seed. This button should be disabled any time song metadata is not available, e.g., during commercials or talk segments.

When the user clicks the Create Station button, the app should display a modal which reads "[seed song name] Radio has been created. Would you like to listen now?"

If the user clicks the Create Custom Station button during a song for which we can not create a custom station, the app should display a temporary message reading "Sorry, we cannot create a custom station from this song.

Thumbs Up & Thumbs Down Buttons

When a user hits the Thumbs Up button the app should display the following message "***" and the Thumbs Up button should remain highlighted in some way so as to indicate that the song has been thumbed up. When the next song in the queue has begun playing the Thumbs Up button should cease to be highlighted.

When a user hits the Thumbs Down button the app should display the following message "***" and the Thumbs Down button should be highlighted in some way. If possible, the app should also advance to the next song in the queue. Once the next song has begun playing, the thumbs down button should cease to be highlighted.

Buy Song Button

Buy Song should be available on any platform that has a mechanism for purchasing music digitally.
