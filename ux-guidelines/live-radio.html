Live Radio

The Live Radio screen

The Live Radio section is a browsable catalog of all 1,500+ terrestrial radio streams that are available on iHeartRadio. Live Radio stations should be browsable by genre (also called format) and location with an emphasis on stations that are geographically near the user.

Popular

Highly popular stations can be featured in this section.

By Location

Stations that are near the user's geographic location should be highlighted in their own subsection. Stations should also be browseable by State and City. All station listings should be sorted according to CUME. This order will be provided by the iHeartRadio servers.

State sections may contain a list of cities or a list of all stations in that state. If using a station list within a state section, please exclude any National Digital Stations.

City sections should display National Digital stations in a seperate subsection at the end of the list of stations.

Station listings should contain the following information:

  • Station Logo
  • Station Name
  • Station Positioner (also called Descriptor)
  • Station Genre (also called Format)
  • Station Location - this should be in the form of [city], [2 letter state]

On platforms with limited display sizes Station listings are required to have the following

  • Station Logo
  • Station Name
  • Station Genre (this is specifically for station listings within Stations by Location)

By Genre

Where appropriate, station genres should be divided into "super-genres". These include Talk Radio, and Music & Entertainment.

Station listings should contain the following information:

  • Station Genre (also called Format)

  • Station Logo
  • Station Name
  • Station Location (this is specifically for station listings within Stations by Genre)
