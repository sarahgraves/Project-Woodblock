Information Architecture

The architecture of the app and how it is navigated will be highly contingent on the form factor and UX norms of the platform it's running on. Below is the optimal implementation for iHeartRadio apps.

Where possible, iHeartRadio apps should use our Global User Experience (GUX) structure. This structure ensures that users coming from other platforms can find their way around based on past experience. The GUX structure includes the following components:

Now Playing

This screen is where users will spend the vast majority of their time. It should contain all required controls for both Live and Custom audio streams and all required metadata. Additional details and content that can make the experience more engaging are encouraged. You can find out more about Now Playing requirements here.

Home Screen

The Home screen is the primary location for merchandised content that our programming team is responsable for populating. It is also responsible for communicating the primary value proposition of iHeartRadio. You can find out more about Home Screen requirements here.

Live Radio

The Live Radio section is a browsable catalog of al 1,500+ terrestrial radio streams that are available on iHeartRadio. This section may also include featured/popular Live Radio stations. You can find out more about Live Radio requirements here.

Create Station

The primary purpose of this sectino is to educate users in creating a Custom Music or Talk station. In addition this section serves as a browsable catalog of all Custom Music, Custom Talk, and iHeartRadio Originals stations. You can find out more about Create Station requirements here.

Search

Search functionality should allow users to quickly find any radio station, artist, album, song, or radio show, using parameters ranging from zip code to name. You can find out more about Search functionality requirements here.

Favorites

The Favorites section should include a user profile; favorite stations, artists, albums, songs, radio shows; and the user's friends. You can find out more about the Favorites section here.

Options

The Options section of the app will contain Account Settings, legal information, and about documents. In some cases this content can be broken into separate sections. You can find out more about the Options section here.
